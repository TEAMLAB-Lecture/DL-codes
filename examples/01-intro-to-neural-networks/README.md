# 신경망 입문

이 예제는 딥러닝의 기본 개념인 신경망(Neural Network)에 대한 소개를 제공합니다.

## 학습 내용

- 신경망의 기본 구조
- 활성화 함수
- 손실 함수
- 경사 하강법
- 간단한 분류 문제 해결

## 실행 방법

```bash
# 예제 디렉토리로 이동
cd examples/01-intro-to-neural-networks

# Docker 컨테이너 실행
docker-compose up -d

# 브라우저에서 접속: http://localhost:8889
```

## 파일 구조

- `notebook.ipynb`: 실습 노트북
- `Dockerfile`: 예제별 Docker 환경 설정
- `docker-compose.yml`: 예제별 실행 설정
